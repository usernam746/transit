# 🚃 交通費変換ツール

モバイルSuica / PASMO の利用履歴を、会社申請フォーマットに自動変換するブラウザツール。

## 🌐 オンラインで使う

ダウンロード不要、ブラウザだけで動きます。

- **トップページ:** https://usernam746.github.io/transit/
- **SUICA版:** https://usernam746.github.io/transit/suica.html
- **PASMO版:** https://usernam746.github.io/transit/pasmo.html

## ✨ 主な機能

- モバイルSuica/PASMOの履歴PDFを自動解析
- 履歴のコピペからも読み込み可能
- 通勤・移動・帰宅を自動分類
- 日別・月別の集計表示
- 会社申請フォーマット（Googleフォーム）への自動転記

## 📖 使い方

### 方法A: PDFから読み込み
1. モバイルSuica/PASMOアプリの会員メニューから「SF残額履歴」を表示
2. PDFを保存し、ツール上の「PDFを選択」ボタンから読み込み
3. 解析されたデータをプレビューで確認、必要なら種別変更
4. 「申請用フォーマット生成」ボタンを押す

### 方法B: 表をコピーして貼り付け
1. 会員メニューの履歴表を範囲選択してコピー
2. ツール上のテキストエリアに貼り付け
3. 「番号順に作業」の手順に従う

## 🔒 プライバシー

**データは全てブラウザ内（localStorage）で処理されます。サーバーへの送信は一切ありません。**
